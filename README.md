# Gerador Cebolão LotoFácil

## Propósito

O Gerador Cebolão LotoFácil é um aplicativo Android desenvolvido para auxiliar os usuários na geração e gerenciamento de suas combinações de jogos para a Lotofácil. A Lotofácil é uma loteria popular no Brasil, onde os jogadores escolhem de 15 a 20 números entre 25 disponíveis. Este aplicativo fornece ferramentas para gerar conjuntos aleatórios, salvá-los, filtrar resultados e gerenciar jogos favoritos, com o objetivo de aprimorar a experiência do usuário com a loteria Lotofácil.

## Recursos

### 1. Game Generation

*   **Geração de Números Aleatórios:** Gere combinações aleatórias de números dentro da faixa da Lotofácil (1 a 25).
*   **Conjuntos Personalizáveis:** Escolha a quantidade de jogos a serem gerados de uma só vez.
*   **Quantidade de Apostas:** Escolha quantos números você deseja apostar (15 a 20).

### 2. Game Management

*   **Salvar Jogos:** Salve os jogos gerados para consulta futura.
*   **Favoritos:** Marque jogos específicos como favoritos para acesso rápido.
*   **Listagem de Jogos:** Visualize uma lista de todos os jogos salvos e gerados.

### 3. Filters

*   **Números Pares/Ímpares:** Filtre os jogos gerados com base no número desejado de números pares e ímpares.
*   **Números Primos:** Filtre os jogos com base na inclusão ou exclusão de números primos específicos.
*   **Soma dos Números:** Filtre os jogos com base na soma dos números.
*   **Números Personalizados:** Filtre os jogos com base na inclusão de números específicos.
*   **Números de Exclusão:** Filtre os jogos com base na exclusão de números específicos.

### 4. Settings

*   **Tema do Aplicativo:** Escolha entre temas claros e escuros para o aplicativo.
*   **Limites de Número de Jogos:** Defina os números padrão e máximo de jogos a serem gerados.
*   **Limpar Dados:** Limpe todos os jogos salvos e as configurações do aplicativo.

### 5. User Interface

*   **Navegação Intuitiva:** Interface fácil de usar com navegação clara entre as diferentes seções.
*   **Design Responsivo:** Otimizado para vários tamanhos e orientações de tela.
*   **Tela de Boas-Vindas:** Exibe uma visão geral dos recursos do aplicativo.

### 6. Data Storage

*   **Banco de Dados Local:** Utiliza um banco de dados local para persistir os jogos gerados e as preferências do usuário.
*   **Persistência de Dados:** Os dados são retidos entre as sessões do aplicativo.

### 7. Technical Features

*   **Kotlin:** Desenvolvido usando a linguagem de programação Kotlin para Android.
*   **Jetpack Compose:** Usa Jetpack Compose para construir a interface do usuário de forma eficiente.
*   **Room:** Emprega a biblioteca Room para o gerenciamento do banco de dados.
*   **DataStore:** Usa o DataStore para armazenar as preferências.
*   **Arquitetura Limpa:** Segue as diretrizes da Arquitetura Limpa para melhor manutenibilidade do código.
*   **ViewModel:** Segue as diretrizes do MVVM para melhor tratamento de dados e atualizações de UI.
*   **Multilínguagem:** o app adapta a interface automaticamente para o idioma do sistema do usuário (ex: inglês, espanhol, etc), caso os recursos estejam disponíveis. O idioma padrão é o pt-BR.

### 8. Utils

*   **LotofacilUtils:** Fornece funções úteis para os jogos da Lotofácil.
*   **Utils:** Fornece funções úteis para todo o aplicativo.

## Instruções de Uso

Este aplicativo foi desenvolvido para simplificar e otimizar a geração de jogos para a Lotofácil, permitindo que os usuários criem combinações de números de forma rápida e gerenciem seus jogos de maneira eficiente.

### Como o App Funciona

O app permite que você:

1.  **Gere Jogos Aleatórios:** Defina o número de jogos e gere-os de forma aleatória.
2.  **Aplique Filtros:** Utilize filtros por números pares/ímpares, números primos, soma dos números e mais.
3.  **Salve e Favorita:** Salve seus jogos gerados e adicione-os aos favoritos para acesso rápido.
4.  **Personalize:** Ajuste as configurações do app, incluindo temas e limites de jogos.
5. **Multi Linguagem:** Adapta a lingua do sistema do usuario automaticamente

### Configurando o App

1. **Abra o Aplicativo:** Inicie o Gerador Cebolão LotoFácil.
2. **Explore as Seções:**

*   **Gerar Jogos**: Acesse a tela de geração de jogos.
*   **Jogos Gerados**: Acesse a lista de jogos gerados.
*   **Favoritos**: Acesse a lista de jogos favoritados.
*   **Filtros**: Ajuste os filtros para gerar jogos com critérios específicos.
*   **Configurações**: Customize o app de acordo com suas preferências.

### Dicas

* **Use os Filtros**: Experimente diferentes combinações de filtros para otimizar suas chances.
* **Salve seus Jogos**: Não perca suas combinações favoritas, salve-as para acessar depois.
* **Explore as Configurações**: Personalize o app ao seu gosto, ajustando temas e limites de geração.
* **Verifique as atualizações**: Baixe as novas versões do app, para verificar as implementações.

### Guia Passo a Passo

1.  **Generate Games:**
    *   Navegue até a seção "Gerar Jogos".
    *   Selecione o número desejado de jogos a serem gerados.
    *   Toque no botão "Gerar".

2.  **View Games:**
    *   Vá para a seção "Jogos Gerados" para ver uma lista dos jogos gerados.

3.  **Add to Favorites:**
    *   Na seção "Jogos Gerados", selecione um jogo.
    *   Toque no ícone "Favorito" para adicioná-lo aos seus favoritos.

4. **View favorites:**
    * Vá para a seção "Favoritos" para ver todos os seus jogos favoritos.

5.  **Apply Filters:**
    *   Vá para a seção "Filtros".
    *   Selecione os filtros desejados (por exemplo, par/ímpar, números primos, soma dos números).
    *   Selecione números personalizados e de exclusão.
    *   Toque em "Aplicar Filtros".

6.  **Access Settings:**
    *   Vá para a seção "Configurações" para personalizar temas, limites de jogos e muito mais.
